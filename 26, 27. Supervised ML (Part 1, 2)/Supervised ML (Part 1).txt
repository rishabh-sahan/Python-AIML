Supervised Learning
    Supervised ML trains algorithms on labeled datasets, meaning each input has a known correct output, to learn patterns and predict outcomes for new data.

1) Data - Preprocessing, Cleaning, Feature Engineering
2) Select Model - Select which Algorithm to use
3) Train & Test - Split the data into the training data & testing data 
    Ex: Random 80% data is used to train the model, and in the other 20% data is used to test the trained model, particular Hypothesis or Relation
4) Underfit or Overfit

Types of problems:
    Regression - output is a continuous numerical value
        Ex: Weather, AQI, House price prediction, Stock price, Score, Stock Price Predicition 
    
    Classification - output is a category or class
        Binary classification -> only two types of output is possible
            Ex: Email (spam or not spam), X-Ray (benign (not-harmful) / malignant (harmful) ). 
        Multi-class classification -> above two types of categories which are all the possible outputs
            Ex: Photo (cat/dog/rabbit), 0-9 Digits.
        Multi-label classification

Linear Regression
Best Fit Line
    error (residual error) = actual value - predicted value
y = c + mx
Theta of straight line
    hθ(X) = θ0 + θ1 X1
        hθ(X) -> hypothesis function
        θ0 -> bias / intercept
        θ1 -> 1st parameter / 1st coefficient
        X1 -> 1st feature
    X1, X2, X3, ...... Xn -> n input features
    hθ(X) = θ0 + θ1 X1 + θ2 X2 ...... θn Xn

Cost Function
    Measures how well the model's predictions match the actual data

Mean Squared Error (MSE)
J(θ) = 1/2*m ∑^m i=1 (hθ(X^(i) - y^(i) )^2

data points = (1,2)(2,4)(3,6)

Gradient Descent
    Gradient Descent is an iterative optimization algorithm used to minimize a cost function by adjusting model parameters in the direction of the steepest descent of the functions gradient.

Summary:
    θ0, θ1 - Initial values
    Make predictions + calculate J(θ)
    Convergence theorem to update parameters

Evaluation Metrics
    Evaluation Metrics are how we measure how good a model is.
Mean Absolute Error (MAE)
    1/n ∑^n i = 1 |yi - y^i|

Mean Squared Error (MSE)
    1/n ∑^n i = 1 |yi - y^i| ^ 2

Root Mean Squared Error (RSME = √MSE)
