Probability
    Probability is a measure of the likelihood that a particular event will occur (Ranges from 0 to 1).
Theoretical Probability -> Coin toss, die or dice roll
Empirical (Experimental) Probability -> 

0 - 1
    0 - Impossible Event
    1 - Certain Event

Important Terms
    Sample Space (S) - all possible outcomes
    Event (E) - a subset of outcomes
    P(E) - P(E) is the probability of event E
        P(E) = no of favorable outcomes / total no of outcomes
                = n(E) / n(S)

Examples:
    1. We rolled a dice, what is probability of getting 4?
        S = {1,2,3,4,5,6}
        E = {4} -> Favourable
        P(4) = 1/6

    2. A card is drawn at random from a deck of 52 playing cards, what is the probability of drawing a heart?
        P(Heart) = 13/52
                = 1/4 = 0.25    

    3. A coin was tossed twice, what is the sample space & probability that both were tail?
        S = {HT, HH, TT, TH}
        E = {TT}
        P(E) = 1/4 = 0.25

Experiments
    Deterministic -> 2 + 3 = 5
    Probabilistic / Random -> Coin toss, Dice roll, Cards drawn

Types of Events
    Certain Event: An event that always happens.
    Impossible Event: An event that cannot happen.
    Probability of the entire sample space S is 1: P(S) = 1
    Probability of an impossible event (no outcomes) is 0: P(Φ) = 0

Intersection & Union Events
    P(A ∩ B) - Probability that A and B both occur. [intersection]
    P(A ∪ B) - Probability that A or B both occur (i.e. atleast one) [union]

Types of Events:
    Mutually Exclusive (Disjoint) Events: Two events that cannot happen at the same time.
        P(A ∩ B) = 0
    Non-Mutually Exclusive Events
    Independent Events: The outcome of one event does not affect the probability of the other.
        P(A ∩ B) = P(A) * P(B)
    Dependent Events: The outcome of one event affects the probability of the other.        
    Complementary Events: Complement event is if one occurs exactly when the other does not.

Note:
    OR - ∪ (Union)
    AND - ∩ (Intersection)

Rule1 - Complementary Rule
    If A is an event, then its complement A^c (event not happening) has probability:
        P(A^c) = 1-P(A)

Rule2 - Addition Rule
    Mutually exclusive events
        If A and B cannot happen together: P(A or B) = P(A) + P(B)
            P(A ∪ B) = P(A) + P(B)
    Non-Mutually exclusive events
        If A and B can happen together: P(A or B) = P(A) + P(B) - P(A ∩ B)
            P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Examples:
    1. Roll a die. Event A is rolling a 2, event B is rolling a 4. Compute P(A ∪ B).
        P(A ∪ B) = P(A) + P(B)
            S = {1,2,3,4,5,6}
            A = {2} = 1/6
            B = {4} = 1/6
                1/6 + 1/6 = 2/6 = 1/3

    2. Roll a die. Event A is we get an even number, event B is getting a multiple of 3. Compute P(A ∪ B) -> (Non-Mutually Exclusive Events)
        P(A c B) = P(A) + P(B) - P(A ∩ B)
            A = {2,4,6}
            B = {3,6}
                3/6 + 2/6 - 1/6
                4/6 = 2/3

Rule3 - Multiplication Rule
    Independent events
        If A and B are independent (one doesn't affect the other): P(A and B) = P(A) * P(B)
            P(A ∩ B) = P(A) * P(B)

    Dependent events
        If A and B are dependent (affect each other): P(A and B) = P(A) * P(B | A)
            P(A ∩ B) = P(A) * P(B | A)
                                P(B | A) -> Probability of event B occuring, Assuming that A has already occured.

Examples: 
    1. Roll a die and flip a coin. What is the probability that we roll a 6 & get a Head? -> (Independent Events)
        P(A ∩ B) = P(A) * P(B)
        P(6 ∩ H) = P(6) * P(H) 
            1/6 * 1/2 = 1/12

    2. A bag contains 5 balls with numbers 1,2,3,4,5. Two are drawn without replacement. What is the probability that first is even & second is also even? -> (Dependent Events)
        P(A ∩ B) = P(A) * P(B | A)
        2/5 * 1/4 = 2/20 = 1/10

Conditional Probability
    It is the probability of an event B occuring given that another event A has already happened.
        We write it as: P(B | A) = P(A ∩ B) / P(A)
            P(B | A) -> Probability of B given given A already occured

Example:
    In a class of 100 students:
        40 students study math.
        30 students pass the math exam.
        25 students both study math and pass the exam.
    What is the probability that a student passes the exam given that they studied math?
        A = Studied Math
        B = Passes exam
            P(B/A) = P(A ∩ B)/P(A)
                   = 25/100 / 40/100
            P(B/A) = 25/40 = 5/8

Law of Total Probability
    It is used to find the probability of an event B by breaking it into several mutually exclusive cases.
    P(B) = P(B | A1) * P(A1) + P(B | A2) * P(A2) +....+ P(B | An) * P(An)

    Where:
        A1, A2,.....An are mutually exclusive and exhaustive events (one of them must happen).
        P(B | Ai) is the probability of B given Ai occurred.

Example:
MachineA: Production Share - 60% Defective Rate - 2%
MachineB: Production Share - 40% Defective Rate - 5%
What is the probability that a randomly chosen product is defective?
    B -> Product is defective
   / \
  A   B 
A = MachineA
B = MachineB
    P(D) = P(D/A)*P(A) + P(D/B)*P(B)
    P(A) = 0.6
    P(B) = 0.4
    P(D/A) = 0.02
    P(D/B) = 0.05
        P(D) = P(D/A)*P(A) + P(D/B)*P(B)
             = 0.02*0.6 + 0.05*0.4
             = 0.012 + 0.020
             = 0.032 =~ 3.2%

Bayes' Theorem
    P(A | B) = (P(B | A) * P(A)) / P(B)
Where:
    P(A | B) = probability of event A given event B occurred
    P(B | A) = probability of event B given event A occurred
    P(A) = probability of event A before knowing B
    P(B) = total probability of event B

Example:
You have an email filtering system:
20% of all emails are spam -> P(spam) = 0.2
The filter correctly identifies spam 90% of the time - P(filter says spam | spam) = 0.9
The filter false flags non-spam emails 5% of the time - P(filter says spam | not spam) = 0.05
If the filter marks an email as spam, what is the probability that it is actually spam ?

Event A -> Actually spam email
Event B -> Marks an email as a spam

    P(spam / filter says spam) = P(A/B)
    P(A/B) = (P(B/A) * P(A)) / P(B)
           = 0.9 * 0.2 / 0.22
          =~ 0.818 ~ 81%
P(B) = P(B/E1)*P(E1) + P(B/E2)*P(E2)
    = 0.9*0.2 + 0.05*0.8
    = 0.18 + 0.4 = 0.22

Random Variable
    A random variable (RV) is a numerical quantity that depends on the outcome of a random experiment.
    Discrete RV - countable values (integer) -> Coin toss {H, T}, Dice
    Continuous RV - any value in a range (fractions) -> Weight, Height, Income

Other Conceepts
Mean: Average value (mu) = sum of values / count of all valus
Median: Middle value 
Mode: Most Frequent value
Weighted Mean -> Some values contribute more than otheer values. Ex: CGPA, GPA

Using Variance & Standard Deviation we measure how spread out the numbers are from the mean.
Variance (σ)^2 = ∑ (xi - mu)^2 / n
√var = σ = Standard Deviation = √∑(xi - mu)^2 / n

∑(xi - mu) / n -> Mean absolute deviation

Probability Distributions
    A probability distribution describes how probabilities are assigned to all possible outcomes.
    It tells you how likely different outcomes are.
Types:
    Discrete Probability Distribution - Discrete RV i.e. outcomes we can count. -> Coin Toss, Dice Ro
        Eg: Binomial Distribution
    Continuous Probability Distribution - Continuous RV i.e. outcome can be any number in range.
        Eg: Uniform Distribution, Normal Distribution
