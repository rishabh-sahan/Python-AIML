Probability
    Probability is a measure of the likelihood that a particular event will occur (Ranges from 0 to 1).
Theoretical Probability -> Coin toss, die or dice roll
Empirical (Experimental) Probability -> 

0 - 1
    0 - Impossible Event
    1 - Certain Event

Important Terms
    Sample Space (S) - all possible outcomes
    Event (E) - a subset of outcomes
    P(E) - P(E) is the probability of event E
        P(E) = no of favorable outcomes / total no of outcomes
                = n(E) / n(S)

Examples:
    1. We rolled a dice, what is probability of getting 4?
        S = {1,2,3,4,5,6}
        E = {4} -> Favourable
        P(4) = 1/6

    2. A card is drawn at random from a deck of 52 playing cards, what is the probability of drawing a heart?
        P(Heart) = 13/52
                = 1/4 = 0.25    

    3. A coin was tossed twice, what is the sample space & probability that both were tail?
        S = {HT, HH, TT, TH}
        E = {TT}
        P(E) = 1/4 = 0.25

Experiments
    Deterministic -> 2 + 3 = 5
    Probabilistic / Random -> Coin toss, Dice roll, Cards drawn

Types of Events
    Certain Event: An event that always happens.
    Impossible Event: An event that cannot happen.
    Probability of the entire sample space S is 1: P(S) = 1
    Probability of an impossible event (no outcomes) is 0: P(Φ) = 0

Intersection & Union Events
    P(A ∩ B) - Probability that A and B both occur. [intersection]
    P(A ∪ B) - Probability that A or B both occur (i.e. atleast one) [union]

Types of Events:
    Mutually Exclusive (Disjoint) Events: Two events that cannot happen at the same time.
        P(A ∩ B) = 0
    Non-Mutually Exclusive Events
    Independent Events: The outcome of one event does not affect the probability of the other.
        P(A ∩ B) = P(A) * P(B)
    Dependent Events: The outcome of one event affects the probability of the other.        
    Complementary Events: Complement event is if one occurs exactly when the other does not.

Note:
    OR - ∪ (Union)
    AND - ∩ (Intersection)

Rule1 - Complementary Rule
    If A is an event, then its complement A^c (event not happening) has probability:
        P(A^c) = 1-P(A)

Rule2 - Addition Rule
    Mutually exclusive events
        If A and B cannot happen together: P(A or B) = P(A) + P(B)
            P(A ∪ B) = P(A) + P(B)
    Non-Mutually exclusive events
        If A and B can happen together: P(A or B) = P(A) + P(B) - P(A ∩ B)
            P(A ∪ B) = P(A) + P(B) - P(A ∩ B)

Examples:
    1. Roll a die. Event A is rolling a 2, event B is rolling a 4. Compute P(A ∪ B).
        P(A ∪ B) = P(A) + P(B)
            S = {1,2,3,4,5,6}
            A = {2} = 1/6
            B = {4} = 1/6
                1/6 + 1/6 = 2/6 = 1/3

    2. Roll a die. Event A is we get an even number, event B is getting a multiple of 3. Compute P(A ∪ B) -> (Non-Mutually Exclusive Events)
        P(A c B) = P(A) + P(B) - P(A ∩ B)
            A = {2,4,6}
            B = {3,6}
                3/6 + 2/6 - 1/6
                4/6 = 2/3

Rule3 - Multiplication Rule
    Independent events
        If A and B are independent (one doesn't affect the other): P(A and B) = P(A) * P(B)
            P(A ∩ B) = P(A) * P(B)

    Dependent events
        If A and B are dependent (affect each other): P(A and B) = P(A) * P(B | A)
            P(A ∩ B) = P(A) * P(B | A)
                                P(B | A) -> Probability of event B occuring, Assuming that A has already occured.

Examples: 
    1. Roll a die and flip a coin. What is the probability that we roll a 6 & get a Head? -> (Independent Events)
        P(A ∩ B) = P(A) * P(B)
        P(6 ∩ H) = P(6) * P(H) 
            1/6 * 1/2 = 1/12

    2. A bag contains 5 balls with numbers 1,2,3,4,5. Two are drawn without replacement. What is the probability that first is even & second is also even? -> (Dependent Events)
        P(A ∩ B) = P(A) * P(B | A)
        2/5 * 1/4 = 2/20 = 1/10

    A single card is drawn from a standard deck of 52 cards.
    Let A = event that the card is a face card (Jack, Queen, King).
    Let B = event that the card is a heart.
    Find P(A ∪ B). -> (Non-Mutually Exclusive Events)
        P(A) = 12/52
        P(B) = 13/52
            P(A ∪ B) = P(A) + P(B) - P(A ∩ B)
                    = 12/52 + 13/52 - 3/52
                    = 25-3/52 = 22/52 = 11/26

Conditional Probability
    It is the probability of an event B occuring given that another event A has already happened.
        We write it as: P(B | A) = P(A ∩ B) / P(A)
            P(B | A) -> Probability of B given given A already occured

Example:
    In a class of 100 students:
        40 students study math.
        30 students pass the math exam.
        25 students both study math and pass the exam.
    What is the probability that a student passes the exam given that they studied math?
        A = Studied Math
        B = Passes exam
            P(B/A) = P(A ∩ B)/P(A)
                   = 25/100 / 40/100
            P(B/A) = 25/40 = 5/8
